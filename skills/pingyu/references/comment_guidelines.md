# 学生评语写作规范

## 基本原则

- 先肯定：优先指出学生已有的优点、进步和闪光点。
- 再建议：不足要委婉表达，强调“下一步可以怎样做”。
- 有依据：尽量结合课堂习惯、作业表现、知识掌握情况，不写空泛套话。
- 有温度：面向学生和家长阅读，避免冷冰冰的分数堆砌。

## 推荐结构

1. 总体评价：学习态度、整体表现或最近进步。
2. 优势描述：知识掌握、思维品质、表达习惯、合作表现等。
3. 成长建议：点明一到两项关键改进方向。
4. 鼓励结尾：表达期待，强化信心。

## 用语建议

- 建议使用：
  - “能够……”
  - “逐步展现出……”
  - “如果能在……方面再主动一些，进步会更明显”
  - “期待你在下阶段……”
- 避免使用：
  - “差”“懒”“不用心”等直接否定标签
  - 多个学生完全相同的套话
  - 只写分数，不写成长表现

## 个性化来源

- 学业：得分率、掌握度高低、薄弱知识点
- 习惯：课堂专注、作业及时、订正态度、表达主动性
- 变化：相比前阶段是否更自信、更认真、更稳定
