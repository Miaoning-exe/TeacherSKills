# 语文出题模板

## 通用要求

- 题目应紧扣文本，不做无依据推断。
- 阅读理解、文言文题建议提供 `material`。
- 解析应说明“文本依据 + 答题角度 + 表达规范”。
- 作文题的答案应是评分标准，不把某篇范文作为唯一答案。

## 难度控制

| 难度 | 特征 |
|------|------|
| 易 | 信息提取、词句理解、基础积累 |
| 中 | 内容概括、人物形象、表达效果 |
| 难 | 主题探究、写法分析、迁移评价 |

## 模板

### 阅读理解

提供短材料，问题围绕内容概括、句子赏析、人物形象、主旨理解展开。

### 古诗词

题目围绕意象、情感、炼字和表现手法，答案要解释诗句含义和表达效果。

### 文言文

题目围绕实词、虚词、句式、翻译和内容理解，解析需要说明语境。

### 作文

题干给出主题、任务、字数和文体边界，答案提供评分要点。
