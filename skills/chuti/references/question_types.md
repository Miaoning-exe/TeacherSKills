# 题型定义

## 通用题型

| 题型 | 适用学科 | 说明 |
|------|----------|------|
| 选择题 | 语文、数学、英语 | 必须提供 `options`，答案使用选项编号或完整选项文本 |
| 填空题 | 语文、数学、英语 | 答案应尽量唯一，必要时在解析中说明等价答案 |
| 解答题 | 语文、数学、英语 | 适合过程性作答，解析应给出步骤或评分要点 |
| 判断题 | 语文、数学、英语 | 答案使用“正确/错误”或“对/错” |

## 数学题型

| 题型 | 说明 |
|------|------|
| 计算题 | 关注运算过程和结果，公式用 LaTeX |
| 证明题 | 需要给出逻辑链条、定理依据和结论 |
| 应用题 | 需要把实际情境转化为数学模型 |

## 语文题型

| 题型 | 说明 |
|------|------|
| 阅读理解 | 建议提供 `material`，答案需引用文本依据 |
| 古诗词 | 关注意象、情感、手法和语言表达 |
| 文言文 | 建议提供 `material`，关注实词、虚词、通假字和句意 |
| 作文 | 答案使用评分标准，不提供固定范文作为唯一答案 |

## 英语题型

| 题型 | 说明 |
|------|------|
| 完形填空 | 建议提供 `material` 和 `options` |
| 阅读理解 | 建议提供 `material`，答案需能从文本中定位 |
| 语法填空 | 明确考查语法点，解析说明规则 |
| 书面表达 | 答案使用评分标准 |
| 翻译 | 解析说明关键词、语法结构和表达习惯 |
