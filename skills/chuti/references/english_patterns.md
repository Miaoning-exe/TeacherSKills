# 英语出题模板

## 通用要求

- 语境自然，避免中式英语。
- 题干和材料词汇难度应匹配年级。
- 解析需要说明语法规则、上下文线索或表达习惯。
- 书面表达题使用评分标准，不提供唯一答案。

## 难度控制

| 难度 | 特征 |
|------|------|
| 易 | 单一词汇或语法点，句子结构简单 |
| 中 | 需要结合上下文判断 |
| 难 | 语篇理解、推断、复杂语法或开放表达 |

## 模板

### 完形填空

提供短语篇和选项，干扰项应覆盖时态、词性、搭配或语义误区。

### 阅读理解

材料应有明确主题，题目覆盖细节理解、主旨大意和简单推断。

### 语法填空

明确考查动词形式、从句、介词、连词或固定搭配。

### 书面表达

给出写作任务、字数、内容要点和评分标准。

### 翻译

关注关键词、句型结构和自然表达，解析说明可接受变体。
