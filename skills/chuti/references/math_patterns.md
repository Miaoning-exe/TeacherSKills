# 数学出题模板

## 通用要求

- 公式使用 LaTeX：行内公式 `$...$`，独立公式 `$$...$$`。
- 题干必须明确已知条件和求解目标。
- 解析应包含关键公式、代入过程和结论。
- 应用题必须说明变量含义，避免脱离学生生活经验。

## 难度控制

| 难度 | 特征 |
|------|------|
| 易 | 单一知识点，直接套用公式或定义 |
| 中 | 两步以上推理，涉及简单综合 |
| 难 | 多知识点综合，要求建模、分类讨论或证明 |

## 模板

### 选择题

围绕 `{knowledge_point}` 判断概念、图像性质、计算结果或解题方法。错误选项应对应常见误区。

### 计算题

给出可验算表达式，答案必须可化简，并在解析中列出中间步骤。

### 证明题

题干应提供足够条件，解析按“已知 -> 推理依据 -> 结论”的结构展开。

### 应用题

情境应真实，变量、单位和目标清晰，解析说明模型建立过程。
