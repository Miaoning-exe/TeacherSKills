# Bloom 分类法参考

| 层级 | 含义 | 课堂常见表现 |
|------|------|--------------|
| 识记 | 回忆事实、概念、规则 | 能复述定义、默写公式、辨认术语 |
| 理解 | 用自己的话解释内容 | 能解释概念、比较异同、说明原因 |
| 应用 | 在熟悉情境中运用方法 | 能完成典型练习、进行句型转换、套用步骤 |
| 分析 | 拆解结构、辨析关系 | 能比较条件变化、分析人物形象、定位解题依据 |
| 评价 | 基于标准做判断 | 能评价观点、判断方法优劣、给出理由 |
| 创造 | 综合迁移并生成新方案 | 能完成开放任务、设计问题、迁移到新情境 |

## 使用建议

- `beike` 报告会根据课标条目中的 `bloom_levels` 生成认知层次分析。
- 备课时应优先保证低层级目标落实，再逐步设计高层级任务。
- 若同一主题覆盖多个层级，教学活动要体现“示范 -> 练习 -> 迁移”的递进。
